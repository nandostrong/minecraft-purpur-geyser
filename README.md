# Servidor Minecraft Purpur + Geyser + Floodgate (24/7)

Servidor completo feito para rodar Java + Bedrock juntos no Koyeb.

## Deploy no Koyeb
1. Acesse https://www.koyeb.com
2. Clique em "Create App"
3. Escolha "GitHub Repository"
4. Selecione este repositório
5. Deploy automático

## Portas necessárias
- Java: 25565/TCP
- Bedrock: 19132/UDP
